## 如何运行

1. 克隆这个仓库：

```bash
git clone https://github.com/Cloudkkk/one-click-shout.git
```

2. 进入项目目录：

```bash
cd one-click-shout
```

3. 安装依赖：

```bash
npm install
```

4. 运行项目：

```bash
npm run tauri
```
